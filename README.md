Log Analyzer Tool

Herramienta desarrollada en Python para analizar archivos de logs y generar reportes automáticos en formato CSV.

---

## 🚀 Descripción

Este proyecto permite procesar múltiples archivos de logs en una carpeta, identificando y contabilizando eventos de tipo:

- ERROR
- WARNING
- INFO

Genera un reporte estructurado con fecha y resumen total, útil para monitoreo y diagnóstico de sistemas.

---

## ⚙️ Tecnologías utilizadas

- Python 3
- Linux (Ubuntu)
- CLI (Command Line Interface)
